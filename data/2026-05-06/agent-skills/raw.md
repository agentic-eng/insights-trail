## FETCHED: OX Security — MCP Architecture Vulnerability

### CVEs Issued
- CVE-2025-65720 (GPT Researcher), CVE-2026-30623 (LiteLLM), CVE-2026-30624 (Agent Zero)
- CVE-2026-30618 (Fay), CVE-2026-33224 (Bisheng/Jaaz), CVE-2026-30617 (Langchain-Chatchat)
- CVE-2026-30625 (Upsonic), CVE-2026-30615 (Windsurf), CVE-2026-26015 (DocsGPT)

### Blast Radius
- 150M+ downloads affected
- 7,000+ exposed servers
- ~200,000 vulnerable instances
- Commands executed on 6 live production platforms

### Affected Tools
LiteLLM, LangChain, IBM LangFlow, Cursor, Windsurf, VS Code, Claude Code, Gemini-CLI

### Anthropic Response
Declined to modify protocol architecture, called behavior "expected," placed responsibility on developers.

### Key Quote
"This is not a traditional coding error...an architectural design decision baked into Anthropic's official MCP SDKs across every supported programming language."

## FETCHED: Check Point Research — CVE-2025-59536 & CVE-2026-21852

- **CVE-2025-59536** – MCP user consent bypass; hooks in `.claude/settings.json` execute before trust dialog
- **CVE-2026-21852** – API key exfiltration via malicious `ANTHROPIC_BASE_URL`

### Attack Vectors
- Supply chain attacks via malicious PRs or honeypot repositories
- Configuration injection into enterprise codebases
- Stolen API keys grant read/write to shared team files

### Status: All patched by Anthropic prior to February 2026 publication

### Key Quote
"The line between configuration and execution continues to blur, requiring us to treat project setup files with the same careful attention we apply to executable code."
