## ARCHITECTURE SUMMARY (from alexop.dev)

"Claude Code is a framework for orchestrating AI agents, and most people use one or two features without seeing how they stack together."

Layer hierarchy:
1. MCP — provides external connections (GitHub, DBs, APIs, browsers)
2. Skills — define workflows and procedural knowledge (30-50 tokens each)
3. Subagents — handle isolated work in separate context windows
4. Hooks — enforce automatic behaviors via event-driven JSON handlers

"The most capable 2026 agents will not pick one but they will combine all three seamlessly depending on the task." (Anand Topu, Medium)

Three-pillar 2026 connectivity stack: Skills (portable capability files) + CLI/Computer Use (local sandbox) + MCP (rich semantics for long-running platform-independent work)
